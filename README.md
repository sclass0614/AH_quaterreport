# 우리집 일상활동 분석 보고서

이 프로젝트는 우리집 데이케어센터의 회원별 활동 데이터를 시각화하여 분석하는 웹 기반 보고서입니다.

## 주요 기능

- Supabase 기반 데이터 관리 및 시각화
- 대분류별 활동 분포 시각화 (최근 3개월 및 이전 3개월 비교)
- 소분류별 활동 빈도 시각화 (막대 차트)
- 대분류/소분류별 참여도, 만족도, 수행도 비교
- 월별 최선호 활동 코드 분석 (최근 6개월)
- 상세 데이터 테이블 제공
- 보고서 인쇄 기능 (선택 회원 또는 전체 회원)
- 인쇄 시 상세 데이터 포함 여부 설정 기능

## 사용 방법

1. 보고서 월 선택: 연도와 월을 선택합니다.
2. 조회: 선택한 월을 기준으로 데이터를 조회합니다.
3. 회원 선택: 드롭다운에서 특정 회원 또는 전체 회원을 선택합니다.
4. 인쇄:
   - '프린트 인쇄하기(선택회원)': 현재 선택된 회원의 보고서를 인쇄합니다.
   - '프린트 인쇄하기(전회원)': 모든 회원의 보고서를 연속으로 인쇄합니다.
   - '상세 데이터 카드 인쇄 포함' 체크박스를 통해 인쇄에 상세 데이터 포함 여부를 설정할 수 있습니다.

## 데이터 시각화 내용

- **우리집 일상활동 유형**: 각 대분류 영역에서 얼마나 많은 활동이 이루어졌는지 비율로 표시 (이전 3개월과 최근 3개월 비교)
- **우리집 일상활동 세부 내역**: 가장 많이 수행된 소분류 코드 상위 10개를 표시
- **우리집 일상활동 유형별/세부내역별 참여도·만족도·수행도**: 세 가지 주요 지표의 평균값을 비교하고 전체 평균과 비교
- **월별 최선호 일상활동 세부내역**: 최근 6개월 동안 월별 선호 활동 코드와 시간, 비율 표시
- **우리집 어르신 활동 일지**: 날짜, 시간, 활동명 등을 포함한 상세 데이터 테이블

## 대분류 코드 정보

- D1: 학습과지식적용
- D2: 일반적과제와요구
- D3: 의사소통
- D4: 이동
- D5: 자기관리
- D6: 가정생활
- D7: 대인상호작용과대인관계
- D8: 주요생활영역
- D9: 지역,사회,시민생활

## 기술 스택

- HTML5
- CSS3
- JavaScript (ES6+)
- Chart.js 및 ChartJS-Plugin-DataLabels
- Supabase (데이터베이스 및 API)

## 최신 업데이트

- 선택회원/전체회원 인쇄 기능 추가
- 인쇄 시 상세 데이터 카드 포함 여부 설정 기능 추가
- 카드 제목 디자인 개선 (민트색 그라데이션 띠 적용)
- 보고서 제목에 '우리집' 텍스트 강조 디자인 적용
- 인쇄 페이지 레이아웃 및 여백 최적화
- 차트 레이블 크기 및 가독성 개선
- 반응형 디자인 향상 (모든 화면 크기에 최적화)

## 참고사항

이 보고서는 우리집 데이케어센터의 회원 활동 데이터를 기반으로 작성되었습니다. Supabase 데이터베이스에서 데이터를 가져와 분석하며, 월별 데이터 조회 및 회원별 필터링을 지원합니다.
